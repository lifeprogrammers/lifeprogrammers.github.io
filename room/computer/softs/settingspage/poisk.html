Разрешения и журнал

Для предоставления релевантных результатов служба Windows
Search будет выполнять поиск в Интернете, приложениях,
параметрах и файлах. Чтобы изменить список файлов на устройстве,
в которых выполняется поиск, перейдите к Поиск в Windows


Безопасный поиск

При предварительном просмотре веб-страниц служба Windows
Search не загружает результаты из Интернета автоматически, если
они могут включать содержимое для взрослых. Если вы решите
использовать предварительный просмотр результатов из Интернета,
мы будем применять приведенный ниже параметр.

Строгий: удалять из результатов веб-поиска текст, изображения
и видео для взрослых




Умеренный: удалять из результатов веб-поиска изображения и
видео для взрослых, но не текст




Выкл.: не удалять содержимое для взрослых из результатов веб-поиска




Поиск содержимого в облаке

Служба Windows Search может персонализировать результаты
поиска путем включения вашего содержимого из OneDrive,
SharePoint, Outlook, Bing и других служб


Учетная запись Майкрософт

Разрешить службе Windows Search предоставлять результаты из
приложений и служб, в которые вы вошли с помощью учетной
записи Майкрософт.


Рабочая или учебная учетная запись

Разрешить службе Windows Search предоставлять результаты из
приложений и служб, в которые вы вошли с помощью рабочей или
учебной учетной записи.

Управление учетными записями, которые используются другими
приложениями на этом устройстве
